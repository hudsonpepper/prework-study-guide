HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown on the webpage.
  • ul stands for "unordered list"
  • ol stands for "ordered list"
  • li stands for 'list item'
  • To add specific characters write &# "ASCII-Code": You can check ASCII codes here
  • To add a link, add < a href="link" > visible text < /a >
  • To add an image, add < img src="filepath" alt="Accessibility Text Description" / >

CSS

  • CSS stands for "Cascading Style Sheets", effectively allowing you to style your < tags >
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside of an element.

Git

Terminal Commands:

  • git status: checks what branch we are working on. Like ls or dir but for git
  • git checkout main: goes to main branch
  • git pull origin main: pulls the latest version of the main branch
  • git checkout -b branch-name: makes a new branch and switches to it
  • git add -A: adds all files to current branch
  • git commit -m "Message for commit"
  • git push origin branch-name: this pushes our changes into the original branch.
  • git remote -v: check current repo

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
